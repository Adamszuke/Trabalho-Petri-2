# Trabalho-Petri-2

Baseado nas definições da estrutura de Pilha, implemente as funções básicas e complementares, assim como foram feitas nas estruturas de Lista e Fila.

Não esquecer as duas versões de implementação, a estática (Simples) e a ligada (Dinâmica).

Faça validações para garantir o uso adequado das estruturas.

- OBS1: Entregas atrasadas serão descontadas
- OBS2: Cópias serão desconsideradas.
- OBS3: Funções/Bibliotecas prontas serão desconsideradas.
- OBS4: Implementações "estranhas" serão supervisionadas. Caso o aluno não saiba explicar o que foi feito, será atribuído nota 0.
- OBS5: A atividade deverá ser feita de maneira individual.
- OBS6: Entrega via repositório público no GitHub.
